pub fn detect_kind(bytes: &[u8]) -> TextKind
Expand description

Decide whether bytes are textual or binary.

A sequence is binary if it contains a NUL byte, is not valid UTF-8, or has too high a ratio of control characters (< 0x20, excluding tab/line feed/ carriage return, plus 0x7F) relative to its total length.
