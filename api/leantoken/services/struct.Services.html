Struct Services 

Source 
pub struct Services { /* private fields */ }
Expand description

Shared application services used by both CLI and MCP adapters.

Blocking filesystem and SQLite work runs on Tokio’s blocking pool. Index reconciliations are serialized across processes, while reads use committed SQLite WAL snapshots so every query in one response sees the same generation.

Return status without initializing an existing SQLite cache.
