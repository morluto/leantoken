Struct Indexer

pub struct Indexer {
    config: Arc<Config>,
    storage: Storage,
    pool: Arc<LazyWorkerPool>,
}
Expand description

Owns discovery/parse publication for one repository cache.

The Rayon worker pool is built lazily on the first non-empty prepare and then reused. Read-only follower processes therefore do not create indexing threads merely by opening repository services.

pub fn reconcile_paths(&self, paths: &[String]) -> Result<IndexResponse>

Reconcile watcher-reported paths without walking the full repository.

Existing regular files and deletions are safe to apply directly. New paths, directories, symlinks, and ignore-rule changes fall back to a full reconciliation because they can affect files beyond the reported path.
